# 💎 Resume Intelligence AI (Platinum Edition)

An advanced, entity-driven talent discovery and semantic coverage engine built with Streamlit and SBERT.

## 🚀 Key Features

- **🧠 Autonomous Learning Mode**: Self-optimizing AI that learns from your Hire/Reject decisions using Stochastic Gradient Descent (SGD) to automatically calibrate ranking weights.
- **🎓 Training & Growth Hub**: Personalized upskilling roadmaps that map technical skill gaps to specific professional courses (Coursera, Udemy, etc.).
- **🎙️ Strategic Interview Studio**: Dynamically generated interview protocols with "Expert Hints" and "Red Flag" indicators based on semantic gap analysis.
- **📊 Intelligence Visualizations**: Dimensional DNA Comparison (Radar charts), Talent Leaderboards, and Skill Distribution heatmaps.
- **🚩 Platinum Gap Analytics**: Distinguishes between critical technical gaps and partial semantic matches (e.g., recognizing "Flask" expertise when "Django" is required).
- **📜 Entity-Discovery Extraction**: Automated parsing of Skills, Work Experience, Education, and Certifications from PDF resumes.

## 🛠️ Technology Stack

- **Core**: Python 3.9+
- **UI Framework**: Streamlit (Glassmorphism UI)
- **Neural Engine**: Sentence-Transformers (SBERT - `all-MiniLM-L6-v2`)
- **Intelligence**: Scikit-Learn (Online Learning), NLTK, PyPDF2
- **Data Visualization**: Plotly Express & Graph Objects

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AnkitRaj027/ResumeAnalyzer.git
   cd ResumeAnalyzer
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   streamlit run app.py
   ```

## 🎯 Usage

1. **Paste Job Specification**: Provide the requirements for the role.
2. **Upload Resumes**: Synchronize PDF resumes to the Talent Pool.
3. **Execute Discovery**: Run the neural engine to rank candidates.
4. **Calibrate**: Use manual sliders or enable **Autonomous Learning Mode** to refine the ranking logic based on your hiring history.
